WebApp Module:
    After cloning the repository, you need to open the terminal at the bottom of IntellIj.
    Inside the terminal run the following commands:
        cd WebApp
        npm install

   Once those complete, you will be able to run the web application with a
   Node JS run configuration on the bin/www Javascript file, the working directory should be the WebApp folder
   and the Environment values should be: DEBUG=webapp:*
